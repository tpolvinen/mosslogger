Readme.txt

Program files to accompany the article, "How to check and calibrate a humidity sensor".

Written for the Quark D2000 Board. Each program simply reads the sensor and sends the reading to the serial monitor.

DHT22.C             Read a DHT-22 sensor
HIH5030.C           Read an HIH5030 sensor
HIH6030.C           Read an HIH6030 sensor
HIH8121.C           Read an HIH8121 sensor
